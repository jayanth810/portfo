Natural Language Processing Model

Natural Language Processing Model to classify Yelp Reviews into 1 star or 5 star categories based off the text content in the reviews and Using the pipeline methods for more complex tasks and another NLP model to predict spam or ham messages. This project basically consists of combining machine learning techniques with text, and using math and statistics to get that text in a format that the machine learning algorithms can understand.
